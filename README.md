## Description

Ce site Internet dédié au Karaté Shotokan de Rungis vise à fournir une plateforme complète pour les amateurs, les pratiquants et les curieux du karaté. Que vous soyez débutant, instructeur chevronné ou simplement intéressé par cet art martial, notre site offre une variété de fonctionnalités et de contenus pour répondre à vos besoins.

## Features

Les fonctionnalités inclues, entre autres :
- La publication des Actualités ;
- La présentation des Activités ;
- L'inscrption en ligne         ;
- La présentation des horaires de cours  ;
- La description bureau gérant actuel    ;
- La présentation des Professeurs        ;
- Les moyens de contacter le club        ;
- La présentation du règlement intérieur ;
- Et les liens relatifs au site internet ;

## Getting Started

Pour déployer une version locale du site, suivez ces instructions :

1. Clonez ce dépôt sur votre machine locale.
2. Exécutez npm install pour installer les dépendances.
3. Lancez l'application avec 'npm start'.
4. Accédez à http://localhost:3000 dans votre navigateur.

### Prerequisites

Pour toute utilisation en locale de ce site internet, utiliser Npm & React en respectant les versions dans les fichiers de configuration.
